The SpamClassifier App

This Spam classifier web app is built on django framework and the styling is done by bootstrap. This is a machine learning project that takes a text data as input and classify it as Spam or Not spam.

Not Spam text messages usually consists messsages from families, friends or from the companies.
Spam messages are mostly advertisement messages.
Try to input any text on your own and see how it works.
